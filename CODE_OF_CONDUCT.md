# Code of Conduct

## Our Commitment

We are committed to a respectful, inclusive, and productive environment for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume positive intent and ask clarifying questions before judging.
- Focus criticism on ideas, code, and process, not people.
- Keep communication clear and evidence-based.
- Support newcomers and make collaboration easier.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Personal attacks or sustained hostile tone.
- Publishing private information without explicit permission.
- Any behavior that makes collaboration unsafe.

## Enforcement

Project maintainers are responsible for enforcing this code.
They may remove, edit, or reject contributions that violate these rules.

## Reporting

Report issues privately to repository maintainers using your preferred secure channel.

## Scope

This Code of Conduct applies in all project spaces:
- repository issues and pull requests,
- discussions and review threads,
- linked community channels when representing the project.
