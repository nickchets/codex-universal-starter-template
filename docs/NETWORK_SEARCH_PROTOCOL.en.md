# Network Search Protocol (EN)

Use network search when:
1. facts are volatile (APIs, versions, rules, prices, policies),
2. user asks for latest/current confirmation,
3. local docs are insufficient,
4. change risk is high.

## Source Priority
1. Official vendor/project docs (primary).
2. Official standards/specifications.
3. Local repository contracts/docs.
4. Secondary sources (blogs/articles) as support only.

## Mandatory Steps
1. Define exact question in one line.
2. Run focused queries (2-4).
3. Open primary sources and confirm key facts.
4. Reconcile with local contracts.
5. Log evidence with URLs and timestamp.

## Evidence Block Template
```md
### Network Cross-Check
- Question: <fact to verify>
- Sources:
  - <url_1>
  - <url_2>
- Checked at (UTC): <timestamp>
- Decision: <accepted conclusion>
- Repo impact: <files/contracts updated>
```
